# FUTURE_CS_02 – Phishing Detection & Awareness System

## 📌 Project Overview
This project focuses on analyzing phishing email attacks and identifying security indicators used in social engineering campaigns.

## 🎯 Objectives
- Identify phishing indicators
- Classify risk level
- Explain attack lifecycle
- Provide prevention guidelines

## ⚠ Risk Classification
PHISHING – CRITICAL

## 🛠 Analysis Approach
- Email content inspection
- Domain pattern analysis
- HTTP vs HTTPS evaluation
- Social engineering assessment
